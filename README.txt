## Project Overview

This is a web-based hotel booking system built using **Python Flask** and **MySQL**. 

## User Features
- User registration and login
- Hotel search by city, dates, and number of guests
- Room selection and booking
- Booking receipt generation
- Booking cancellation with applicable charges
- Password and email update

## Admin Features
- Admin login/logout
- Add, update, and delete hotels
- Update room prices (peak/off-peak)
- Manage room availability
- Update currency exchange rates
- View reports (monthly sales, hotel revenue, top customers)
- Change own password/email
- Reset other usersâ€™ passwords
